# Devam Trivedi - Cybersecurity Portfolio

A high-performance, cybersecurity-themed portfolio website featuring advanced animations, terminal-style interfaces, and interactive sections highlighting my expertise in ethical hacking, networking, and security analysis.

## Features

*   **Cybersecurity Theme**: Matrix-style backgrounds, terminal interfaces, and sleek dark mode UI.
*   **Interactive Components**: Simulated cyber-assistant, project grids, and skill proficiency matrices.
*   **Responsive Design**: optimized for all devices with smooth animations.
*   **Security Focused**: Styled to reflect professional security standards.

## Tech Stack

*   **Frontend**: React, TypeScript, Tailwind CSS
*   **Icons**: Lucide React
*   **Build Tool**: Vite

## Run Locally

1.  Msg me at Devam9131@gmail.com for access or cloning instructions.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```

## Deploy

Build for production:

```bash
npm run build
```
